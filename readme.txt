== Description ==

This plugin was created to add content of your
wordpress blog to the iPad app Blog Reader. With
Blog Reader your blog content will be available to iPad
users. New blogs and sites - large and small, covering
various subjects - debut every day at Blog Reader,
increasing its audience and profitability. Submit your
content now to IPAD and make your blog famous at
this new platform.

= Note: Users of this plugin does not need to register on the site =
For more information: [Blog Reader](http://www.blogreader.com.br)

Este plugin tem como finalidade agregar o conteúdo de seu blog wordpress ao aplicativo
*Blog Reader* feito para iPad. Ele traz a vantagem de distribuir seu conteúdo e coloca-lo ao lado
de grandes portais . Novos sites e blogs - grandes e pequenos, sobre os mais diversos assuntos - 
entram todos os dias para o Blog Reader, aumentando sua audiência e rentabilidade. Envie agora mesmo seu conteúdo para o iPad e marque presença nessa nova plataforma.

= Atenção: Usuários deste plugin não precisam se cadastrar no site =
Para maiores informações acesse: [Blog Reader](http://www.blogreader.com.br)

== Installation ==

Upload the plugin to the /wp-content/plugins/
directory. And activate it. Enter a url of an image of
your website profile, and a style sheet (if you want to customize the final view on the iPad App BlogReader)

*Note:*
This plugin only starts to work after the aproval of you
blog by the Blog Reader team. For more information visit: [Blog Reader](http://www.blogreader.com.br)

Faça upload do plugin para a o diretório /wp-content/plugins/. E ative-o.
Informe uma url para uma imagem de perfil de seu site, e uma folha de estilos (caso queira personalizar a visualização final na App BlogReader)

*Observação:* 
Este plugin só inicia a captura de conteúdo após sua aprovação no Blog Reader.
para maiores informações acesse: [Blog Reader](http://www.blogreader.com.br)

== Frequently Asked Questions ==

= What is Blog Reader? =
The Blog Reader is an application for the iPad, which
aims to aggregate blogs and websites of all kinds of
subjects.

= What does this plugin? =
It makes your content available to Blog Reader
application without further work beyond installing the
plugin.

= How it works? =
Simple. When you post an article, this will be captured
and indexed at the iPad Blog Reader App.

= How can I configure it? =
Click on the Blog Reader tab to access the
configuration page. On this page you can - amongst
other things - determine the category that your blog fits in.

= O que é o Blog Reader? =
O blog Reader é um aplicativo para o iPad, que tem por objetivo agregar blogs e sites
dos mais variados assuntos.

= O que faz este plugin? =
Ele agrega o seu conteúdo ao aplicativo Blog Reader, sem maiores esforços além da instalação
do plugin.

= Como funciona? =
Simples. Quando o usuário postar algum artigo, este será capturado para indexação. 

= Como configurar? =
Clique na aba *Blog Reader* para acessar a página de configuração. Nesta página é possível 
- entre outras coisas - determinar a categoria em que seu blog se enquadra.
